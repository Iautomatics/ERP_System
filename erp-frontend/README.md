# ERP Frontend - Next.js

Interfaz de usuario del sistema ERP desarrollada con Next.js 16, TypeScript y Tailwind CSS.

## Tecnologías

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Recharts

## Instalación

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Variables de entorno

```
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

Ver README principal del proyecto para más información.
